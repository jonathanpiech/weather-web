Summary: X Vs. Latitude

The purpose of this project was to analyze the weather with respect to latitude. The OpenWeatherMap API was used to create a dataset of over 500 cities to complete these analyses. Matplotlib was used to plot the various weather relationships against latitude. Those relationships included: cloudiness, humidity, max temperature, and wind speed. This site provides the data and plots of those relationships, as well as analysis and explanations of those relationships.
